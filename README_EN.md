# Street Life

A modern, urban-style news article website.

This project presents an HTML page containing multiple articles with dates, images, and captions.  
The layout uses semantic tags and a clear structure for an enjoyable and accessible user experience.

---

## Description

- Static site built with HTML5 and CSS3 (external CSS files)  
- Simple navigation menu with placeholder links  
- Three articles featuring dates, images, and descriptive captions  
- Footer containing legal notices and social media icons  
- Designed for readability and user experience

## Main Features

- Use of HTML5 semantic tags (`<header>`, `<nav>`, `<main>`, `<article>`, `<footer>`)  
- Date management with `<time datetime="YYYY-MM-DD">` for better accessibility  
- Images integrated with captions using `<figure>` and `<figcaption>`  
- Accessible menu with `aria-label` attribute  
- Clickable social media icons in the footer

---

## Installation and Usage

1. Clone this repository or download it as a ZIP file.  
2. Ensure the `img/` and `css/` folders are in the same directory as `index.html`.  
3. Open `index.html` with a modern web browser (Chrome, Firefox, Edge, Safari).  
4. Navigate the site to discover the articles.
