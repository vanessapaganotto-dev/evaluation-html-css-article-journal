# Street life

**Site web d’articles de presse au style urbain et moderne**  
Ce projet présente une page HTML contenant plusieurs articles avec dates, images et légendes. La mise en page utilise des balises sémantiques et une structure claire pour un rendu agréable et accessible.

---

## Description

- Site statique en **HTML5** et **CSS3** (avec fichiers CSS externes)  
- Menu de navigation simple avec liens fictifs  
- Trois articles avec dates, images et légendes descriptives  
- Footer contenant mentions légales et icônes sociales  
- Conception pensée pour la lisibilité et l’expérience utilisateur

## Fonctionnalités principales

- Utilisation des balises sémantiques HTML5 (`<header>`, `<nav>`, `<main>`, `<article>`, `<footer>`)  
- Gestion des dates avec la balise `<time datetime="YYYY-MM-DD">` pour une meilleure accessibilité  
- Intégration d’images avec légendes via `<figure>` et `<figcaption>`  
- Menu accessible grâce à l’attribut `aria-label`  
- Icônes sociales cliquables dans le pied de page

---

## Installation et utilisation

1. Clonez ce dépôt ou téléchargez-le en ZIP.  
2. Assurez-vous que les dossiers `img/` et `css/` sont dans le même répertoire que `index.html`.  
3. Ouvrez `index.html` avec un navigateur web moderne (Chrome, Firefox, Edge, Safari).  
4. Naviguez dans le site pour découvrir les articles.
